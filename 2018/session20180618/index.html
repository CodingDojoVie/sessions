Your mission, should you choose to accept it ...

A famous casino is suddenly faced with a sharp decline of their revenues.

They decided to offer Texas Hold'em also online.

Can you help them by writing an algorithm that can rank poker hands?

Ranking Poker Hands

Task

Create a poker hand that has a method to compare itself to another poker hand: 
Result PokerHand.compareWith(PokerHand hand);
A poker hand has a constructor that accepts a string containing 5 cards: 
PokerHand hand = new PokerHand("KS 2H 5C JD TD");
The result of your poker hand comparison can be one of these 3 options: 

public enum Result
{
    WIN,
    LOSS,
    TIE
}
bit.ly/top-poker-hands

String of cards


  • A space is used as the card separator

  • Each card consists of two characters

  • The first character is the value of the card, valid characters are: 
    2, 3, 4, 5, 6, 7, 8, 9, T(en), J(ack), Q(ueen), K(ing), A(ce)

  • The second character represents the suit, valid characters are: 
    S(pades), H(earts), D(iamonds), C(lubs)
