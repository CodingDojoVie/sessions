Round 1

The Plan

Find a partner, agree on language & goal
Refactor this code, take small steps, run the tests often
Proposal: Babysteps - Smallest steps possible
  • TCR - test && commit || revert
  • Run all tests after 1 minute
  • If they fail, you revert
  • If they are green, you commit
  • Switch keyboard after 5 minutes
