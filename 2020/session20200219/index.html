Round 1

The Plan

Find a partner and agree on programming language
Download code: https://github.com/jbrains/trivia
 
Goal: Add current time to printed messages
 
-> Identify 1 line of code you want to change
-> Create (failing) test that executes this line
-> Change production code
-> Refactor test function to not call Game object directly
Motivation: Get people to focus on one particular functionality (one log output statement) per test. Furthermore, force people to make tests readable. Instead of simply exercising the functions of production code, the intent should be made clear. (why is there a call sequence of roll, wrongAnswer, roll, ...) In short: It should become a given, when, then language.
