Go sequentially

-> Implement as simple as possible to get test green
-> Refactor all code to become clean code

Round 2

Baby steps!

Same as before, but:
Baby steps - timed!

  1. Prepare repository
  2. Start timer with 2 minutes.
  3. Write a (red) test, make it green.
  4. If timer runs out: Reset and goto 2.
  5. If not, Commit.
  6. Start timer with 2 minutes.
  7. Refactor
  8. If timer runs out: Reset and goto 6.
  9. If not, Commit and goto 2.

At this point, the people should have a rhythm on doing the exercise. They can start from scratch, or continue where they left off. Recommendation is to switch partners and start fresh.
